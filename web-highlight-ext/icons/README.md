# Icons

Place your icon files here:
- `icon16.png` (16x16)
- `icon48.png` (48x48)
- `icon128.png` (128x128)

You can generate a simple icon from any image using https://favicon.io/favicon-converter/
or just copy any 128x128 PNG and rename it for all three sizes.

Or run `python tools/generate_icons.py` to generate placeholder icons automatically.
